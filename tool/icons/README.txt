Place raster icons here for mobile home‑screen shortcuts.

Required files (recommended sizes):
- apple-touch-icon.png  (180x180)  — used by iOS when “添加到主屏幕”
- icon-192.png          (192x192)  — Android/Chrome manifest icon
- icon-512.png          (512x512)  — Android/Chrome manifest icon
- icon-512-maskable.png (512x512)  — Android maskable icon (safe padding)

Tips:
- Keep important content within a safe area (~80% center) for iOS rounded corners.
- For maskable icon, provide padding so shapes aren’t clipped by adaptive masks.
- PNG with transparent background is preferred; avoid heavy drop shadows.
